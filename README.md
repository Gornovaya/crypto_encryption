# crypto_encryption

Коллекция реализаций классических и современных криптографических алгоритмов на C++ и Python. Учебный проект для демонстрации принципов шифрования, хэширования и криптоанализа.

### c++ работы:
1. Шифры замены: алгоритм шифрования данных «Полибианский квадрат».
2. Шифрование методом гаммирования: алгоритм шифрования данных «Шифрование методом гаммирования» по модулю 2.
3. Режимы работы DES: стандарт шифрования данных DES в режиме «Электронная кодовая книга».
4. Режимы работы DES_2: стандарт шифрования данных DES в режиме в режиме «Обратная связь по выходу».
5. Ассиметричные алгоритмы шифрования: алгоритм шифрования данных «RSA».

### питон работы:
1. Частотный анализ: взломать зашифрованное сообщение с использование частотного крипто анализа. Написать реализацию дешифрования методом частотного анализа – взять исторический шифр и зашифровать сообщение. Выяснить, есть ли связь между длиной зашифрованного сообщения и успешного дешифрования, попробовать дополнительно к частотному анализу найти часто встречающиеся последовательности – союзы или предлоги.
2. Потоковые алгоритмы: реализовать любой из рассмотренных нами потоковых алгоритмов шифрования – RC4.
3. Ассиметричные алгоритмы: алгоритм Меркла-Хеллмана.
4. Хэш-функции: реализовать алгоритм хэширование md5 и проверку, и подстановку ЭЦП.
